# LabStreamLayer

## Summary
Project containing LabStreamLayer components.

## Dependencies
* [lsl_csharp](dep/lsl_csharp.dll) https://github.com/labstreaminglayer/liblsl-Csharp to build in .net framework 4.8
* [lsl.dll](dep/lsl.dll) https://github.com/sccn/liblsl/releases

## Files
* [ILabStreamLayerComponent](src/ILabStreamLayerComponent.cs) interface for LabStreamLayer component.
* [LabStreamLayerComponent](src/LabStreamLayerComponent.cs) LabStreamLayer generic component.
* [LabStreamLayerManager](src/LabStreamLayerManager.cs) component that manage LabStreamLayer connections and streams.
