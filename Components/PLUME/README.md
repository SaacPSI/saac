# PLUME

## Summary
Project handling interop with [PLUME](https://github.com/liris-xr/PLUME), this project is an example to parse data from PLUME storage file to \psi dataset.

## Files
* [Plume Dataset Pipeline](src/PlumeDatasetPipeline.cs) class that manage the dataset and pipeline to record data into PsiStore.
* [Plume File Parser](srcPlumeFileParser.cs) class handling the management of the PLUME file storage.
